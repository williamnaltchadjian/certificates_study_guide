# AWS Documentation: Amazon Q

Este repositório centraliza o conhecimento técnico sobre o Amazon Q e seus diferentes serviços.

---

## Introdução ao Amazon Q

O Amazon Q é o assistente de IA generativa da AWS voltado para:

* **Negócios** – Produtividade corporativa
* **Desenvolvedores** – Auxílio na programação
* **Usuários AWS** – Assistência no console

**Objetivo:** Aumentar produtividade com IA integrada ao ecossistema AWS.

---

## Amazon Q Business

O Amazon Q Business é um assistente de IA para empresas.

### O que faz:
* Responde perguntas com base em dados internos
* Conecta-se a sistemas como SharePoint, Salesforce, S3 etc.
* Usa RAG para buscar informações corporativas

### Foco:
* Produtividade corporativa
* Busca inteligente
* Segurança empresarial

!!! important
    Respeita permissões de acesso (controle de segurança).

---

## Amazon Q Business – Hands On

Na prática:

1. Conectar fontes de dados
2. Indexar documentos
3. Fazer perguntas em linguagem natural
4. Obter respostas com base no conteúdo interno

### Diferencial:
* Respeita permissões de acesso (controle de segurança)

---

## Amazon Q Apps

O Amazon Q Apps permite criar aplicações GenAI sem código.

### Usuários podem:
* Criar pequenos apps internos
* Automatizar tarefas
* Compartilhar apps com equipe

### Foco:
* No-code
* Automação rápida
* Produtividade

---

## Amazon Q Developer

O Amazon Q Developer é um assistente de IA para desenvolvedores.

### Funcionalidades:
* Geração de código
* Explicação de código
* Refatoração
* Sugestões dentro da IDE
* Ajuda com serviços AWS

!!! tip
    Aumenta produtividade de desenvolvimento.

---

## Amazon Q Developer – Hands On

Na prática:

1. Integrado ao VS Code / IDE
2. Gera código automaticamente
3. Explica erros
4. Sugere melhorias
5. Auxilia com comandos AWS CLI

---

## Amazon Q for AWS Services

Integração do Amazon Q diretamente no console da AWS.

### Permite:
* Explicar configurações
* Ajudar na criação de recursos
* Diagnosticar problemas
* Gerar comandos e scripts

!!! note
    Funciona como um "assistente inteligente dentro do console".

---

## PartyRock

O PartyRock é uma plataforma experimental da AWS para criar aplicações GenAI de forma simples.

### Permite:
* Criar apps com prompts
* Testar ideias rapidamente
* Compartilhar aplicações

### Foco:
* Aprendizado
* Experimentação com GenAI
* Interface visual simples

---

## Resumo dos Serviços

| Serviço | Foco |
| :--- | :--- |
| **Amazon Q Business** | IA corporativa com dados internos |
| **Amazon Q Apps** | Criar apps GenAI sem código |
| **Amazon Q Developer** | Assistente para programadores |
| **Q for AWS Services** | Assistente dentro do Console AWS |
| **PartyRock** | Criar apps GenAI de forma simples |
