# AWS Services: Serviços de IA da AWS

---

## Visão Geral dos Serviços

| Serviço | O que faz | Caso de Uso Principal |
| :--- | :--- | :--- |
| **Amazon Comprehend** | Análise de texto (NLP) | Sentimento, entidades, tópicos |
| **Amazon Translate** | Tradução automática | Traduzir conteúdos multilíngues |
| **Amazon Transcribe** | Converte áudio em texto | Transcrição de chamadas |
| **Amazon Polly** | Converte texto em áudio | Assistentes virtuais |
| **Amazon Rekognition** | Análise de imagens e vídeos | Reconhecimento facial, objetos |
| **Amazon Lex** | Criação de chatbots | Bots com voz e texto |
| **Amazon Kendra** | Busca inteligente empresarial | Pesquisa em documentos internos |
| **Amazon Mechanical Turk** | Trabalho humano sob demanda | Rotulagem de dados |
| **Amazon Augmented AI (A2I)** | Revisão humana em workflows de IA | Validação de previsões |
| **Amazon Personalize** | Sistema de recomendação | Recomendar produtos |
| **Amazon Textract** | Extrai texto de documentos | Processar PDFs e formulários |
